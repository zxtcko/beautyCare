注册：
url: http://www.zendaiart.net/beaut/registeruser.php
方式：POST
参数：
mobile	手机号
user	注册帐号
pass	密码
string	校验码（生成：md5(pass.md5(time.user.check_char.mobile))，其中 check_char="QW!!22*&90)" ）
time	时间戳
返回值：（json数据）
iserror	为1时表示错误，0时表示成功
info	错误或成功的信息

商户查询：
url: http://www.zendaiart.net/beaut/orginfosearch.php
方式：POST
参数：
skey	商户关键字
city	城市名称
string	校验码（生成：md5(city.md5("org_name".check_char.skey))，其中 check_char="QW!!22*&90)" ）
time	时间戳
返回值：（json数据）
iserror	为1时表示错误，0时表示成功
info	iserror=1时返回的错误信息
data	iserror=0时返回的数据，格式（数组）：
	info(商户信息)：
	    id: 商户ID
	    name: 商户名称
	    store（门店信息）：
	    	id: 门店ID
	    	name: 门店名称
